Selector

Tag HTML,ID,Class ทีต้องการกำหนดคุณสมบัติ แบ่งออกเป็น 3 รูปแบบ

1. Element: กำหนด style ให้ html โดยตรง
2. Class: กำหนด style โดยใช้ชื่อ class เป็น selector โดยใช้จุด
3. ID: กำหนด style โดยใช้ชื่อ ID เป็น selector โดยใช้เครืองหมาย #

รูปแบบการประกาศใช้ CSS

แบ่งออกเป็น 2 รูปแบบ

1. Internal: แบ่งออกเป็น 2 ชนิด
    1. Inline: แทรกลงใน tag
    2. Embeded: กำหนดบน head เขียนภายใต้ style และ /style

Position

Position แบ่งออกเป็น 5 แบบ

  1. static: default
  2. relative: นับจากจุดนั้นๆที่วัตถุอยู่
  3. absolute: จัดวางวัตถุให้อยู่ตำแหน่งไหนก็ได้แต่ต้องกำหนดจากขอบ element (ไม่มีอะไรครอบถือเป็น body)
  4. fixed: วางวัตถุให้อยูู่ตำแหน่งเดิม
  5. sticky: ให้วัตถุติดขอบเมื่อเลื่อนไปถึง

Advance Selector

Advance Selector แบ่งออกเป็น 5 แบบ

  1. Targeted Selector
    • Descendant Selector
      • การใช้ เช่น div p
    • Child Selector
      • การใช้ เช่น div>p
  2. Style by Attribute
    3. ใช้ Attribute เป็นเกณฑ์ เช่น a กำหนดโดยใช้ target a[target="_blank"]{ color: red; }
  3. Special Attribute
    4. ใช้คล้าย special Attribute แต่กำหนดได้มากกว่า 1 ค่า เช่น input[type="email"],input[type="text"]{ color: red; }
  4. Pseudo Selector
    5. การใช้ให้นำ element มาต่อด้วย : เช่น :first-child,:last-child,:nth-child(2),:nth-child(2n),:nth-child(even),:nth-child(odd)
    • First-Child
    • Last-Child
    • nth-Child
  5. Before & After Pseudo
    6. ใช้เนื้อหาด้านหน้าหรือด้านหลัง element ที่ต้องการทำงาน เช่น header::before
