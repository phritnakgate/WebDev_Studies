DOM

DOM (Document Object Model)คือ HTML ที่มองเป็นก้อน Object เมื่อหน้าเว็บโหลดเสร็จเรียบร้อย Web Browser จะสร้าง DOM ออกมา

คุณสมบัติของ DOM
  • เข้าถึงและเปลี่ยนคุณสมบัติของ Element ในหน้าเว็บได้
  • ควบคุมและเปลี่ยนรูปแบบ CSS
  • ตอบสนองกับทุกเหตุการณ์ที่เกิดขึนในหน้าเว็บได้

เข้าถึง Element ผ่าน ID,Tag,Class

  • document.getElementById(ID);
  • document.getElementByTagName(Tag);
  • document.getElementByClassName(Class);

เปลี่ยนเนื้อหา

  • HTML: element.innerHTML
  • Text: element.innerText
  • style: element.style.properties = value

หรือ element.setAttribute(attribute,value)

ตัวอย่าง

เงินคงเหลือจะปรากฏที่นี่

DOM Nodes

  • สร้าง Element ใหม่: document.createElement(element)
  • ลบ node: document.removeChlid(element)
  • ต่อ element ใน node แม่: document.appendChild(element)
  • แทนที่ element: document.replaceChild(new,old)
ตัวอย่าง

    DOM CSS

    • เพิ่ม class style: element.classList.add("class")
    • ลบ class style: element.classList.remove("class")
    • สลับ class style: element.classList.toggle("class")
    • เปรียบเทียบ class style: element.classList.contains("class")

    DOM Event

    คือ เหตุการณ์หรือการกระทำบางอย่างที่เกิดขึ้นกับ element เช่น ย้ายเมาส์,กดปุ่ม keyboard

    EventListener

    คือ เหตุการณ์หรือการกระทำบางอย่างที่เกิดขึ้นกับ element แต่รูปแบบการเขียนจะเขียนในฝั่ง js
